# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository overview

This is a personal document archive containing materials related to the Italian engineering professional qualification exam (abilitazione all'esercizio della professione di ingegnere). There is no source code, build system, or test suite.

## Contents

- **`abil.pdf`** — main qualification document
- **`dispense/`** — course notes and lecture slides, organized into:
  - `Dispense/` — lecture materials (slides and PDFs on measurement instruments, oscilloscopes, ADCs, etc.)
  - `Misure_Elettriche/` — electrical measurements notes, exercises, and lab reports
- **`documenti/`** — administrative documents (exam schedules, application forms, results from 2010 sessions)
- **`vecchi_compiti/`** — past exam papers (electronics, information engineering)

## File types

Most files are PDFs, PowerPoint presentations (.ppt), and RTF documents. There is no executable code in this repository.
