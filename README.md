# Abilitazione Ingegneria

Materiale per l'esame di abilitazione all'esercizio della professione di Ingegnere dell'Informazione — *Italian Professional Engineering Exam preparation*.

> L'Esame di Stato per ingegnere è richiesto per l'iscrizione all'Albo professionale in Italia.
> Questo repository raccoglie il materiale di studio preparatorio per l'indirizzo **Ingegneria dell'Informazione**.

## Contenuti

| Directory | Descrizione |
|-----------|-------------|
| `dispense/` | Dispense e slide dei corsi di preparazione |
| `dispense/Dispense/` | Strumenti di misura, oscilloscopi, convertitori ADC, contatori, voltmetri, multimetri |
| `dispense/Misure_Elettriche/` | Appunti, esercitazioni, compitini e relazione di Misure Elettriche |
| `documenti/` | Documenti amministrativi — calendario prove, domande, esiti sessioni 2010 |
| `vecchi_compiti/` | Temi d'esame passati (elettronica, ingegneria dell'informazione) |
| `abil.pdf` | Documento principale di sintesi |

## Struttura dell'esame

L'esame di abilitazione si compone di:
1. **Prova scritta** — temi d'esame su elettronica, sistemi, telecomunicazioni, campi
2. **Prova orale** — discussione su discipline dell'indirizzo informazione
3. **Prova pratica** — progetto o relazione tecnica

I `vecchi_compiti/` coprono la prova scritta, mentre le `dispense/` forniscono la preparazione teorica.

## Lingua

Il materiale è interamente in italiano, lingua ufficiale dell'esame.
