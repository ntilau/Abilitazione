# Abilitazione Ingegneria

Materiale per l'esame di abilitazione all'esercizio della professione di ingegnere.

## Contenuti

- **`dispense/`** — Dispense e slide dei corsi
  - `Dispense/` — Strumenti di misura, oscilloscopi, convertitori ADC, contatori, voltmetri, multimetri
  - `Misure_Elettriche/` — Appunti, esercitazioni, compitini e relazione di Misure Elettriche
- **`documenti/`** — Documenti amministrativi (calendario prove, domande, esiti sessioni 2010)
- **`vecchi_compiti/`** — Temi d'esame passati (elettronica, ingegneria dell'informazione)
- **`abil.pdf`** — Documento principale
